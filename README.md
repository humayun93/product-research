# Product Research & Documents

This repository hosts various documents, presentations, and research materials for my projects and experiments.

## Features

- Responsive design that works on all devices
- Modern UI with smooth transitions and animations
- Interactive navigation with keyboard and touch support
- Progress tracking with visual progress bar
- Beautiful typography using Inter font family
- Custom scrollbars for better aesthetics
- Support for embedded videos and images
- Mobile-friendly with swipe gestures

## Technology Stack

- HTML5
- TailwindCSS (via CDN)
- Vanilla JavaScript
- Google Fonts (Inter)
